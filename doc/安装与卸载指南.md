## 1. 环境要求

| 要求 | 说明 |
|---|---|
| **操作系统** | Windows 10+, macOS 12+, Linux（X11/Wayland） |
| **Python** | ≥ 3.10（推荐 3.11+） |
| **pip** | ≥ 23.0 |
| **磁盘** | ~500 MB（含依赖），项目本身 ~300 KB |
| **Git**（可选） | 克隆仓库时使用 |

### 检查 Python 版本

```bash
python --version
# 应输出: Python 3.10.x 或更高
```

如果未安装 Python，请访问 [python.org](https://www.python.org/downloads/) 下载安装。

## 2. 在线安装（标准方式）

### Step 1：获取代码

```bash
# 方式 A：Git 克隆
git clone https://github.com/WEN454545/Py_Project.git
cd Py_Project

# 方式 B：手动下载 ZIP 解压
# 访问 https://github.com/WEN454545/Py_Project
# Code → Download ZIP → 解压到本地
```

### Step 2：创建虚拟环境（推荐）

```bash
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
# Windows:
.venv\Scripts\activate
# macOS / Linux:
source .venv/bin/activate
```

### Step 3：安装

```bash
# 安装应用 + 所有依赖
pip install -e ".[dev]"

# 仅安装运行依赖（不包含测试工具）
pip install -e .
```

**依赖列表（自动安装）：**

| 包 | 用途 |
|---|---|
| PySide6 ≥ 6.6 | Qt GUI 框架 |
| markdown-it-py ≥ 3.0 | Markdown 解析 |
| mdit-py-plugins ≥ 0.4 | 表格、前置元数据插件 |
| PyYAML ≥ 6.0 | YAML Frontmatter 解析 |
| Pygments ≥ 2.17 | 代码语法高亮 |
| Pillow ≥ 10.0 | 截图与图片处理 |
| diff-match-patch | 版本差异对比 |
| lxml ≥ 5.0 | OPML XML 导出 |
| python-slugify ≥ 8.0 | 文件名安全化 |
| bleach ≥ 6.0 | HTML 清洗 |

开发依赖（`[dev]`）：

| 包 | 用途 |
|---|---|
| pytest ≥ 8.0 | 测试框架 |
| pytest-qt ≥ 4.4 | Qt GUI 测试 |
| coverage ≥ 7.0 | 代码覆盖率 |

## 3. 离线安装（网络受限环境）

如果所在网络无法访问 PyPI（如企业防火墙阻断 SSL），以下为离线安装方案。

### Step 1：准备依赖包（在有网机器上）

```bash
# 在可上网的机器上下载所有依赖（包括对应平台的 PySide6）
mkdir pyknowledge_packages
pip download -d pyknowledge_packages -e .
pip download -d pyknowledge_packages pytest pytest-qt coverage

# 将整个 pyknowledge_packages 目录复制到目标机器
```

### Step 2：在目标机器上离线安装

```bash
# 激活虚拟环境后
pip install --no-index --find-links=<path_to_packages> -e .
```

### Step 3：强制 stdlib 验证（不依赖 PySide6）

由于 PySide6 ~120 MB 且依赖 Qt C++ 运行时，下载可能较困难。可以先用纯 Python 测试验证代码逻辑正确：

```bash
# 此命令不依赖 PySide6，仅使用 stdlib
python tests/test_verify_all.py
# 输出: ALL TESTS PASSED [OK] — 表示代码逻辑完整
```

### 平台注意事项

| 平台 | PySide6 Wheel 大小 | 备注 |
|---|---|---|
| Windows AMD64 | ~60 MB | 推荐 Python 3.11 x64 |
| macOS ARM64 | ~80 MB | 需 macOS 12+ |
| Linux x86_64 | ~70 MB | 需 Qt 6 系统库可选 |

## 6. 卸载

### 完全卸载步骤

#### Step 1：卸载 Python 包

```bash
# 如果是 editable install
pip uninstall py_project -y

# 如果在虚拟环境中，也可直接删除虚拟环境目录
# Windows:
rmdir /s .venv
# macOS / Linux:
rm -rf .venv
```

#### Step 2：删除全局安装的依赖（可选）

如果不再需要此项目的依赖用于其他项目：

```bash
pip uninstall -y PySide6 markdown-it-py mdit-py-plugins PyYAML Pygments Pillow diff-match-patch lxml python-slugify bleach
```

#### Step 3：删除应用数据

应用数据存储在平台特定目录（包含所有笔记、标签、附件）：

```bash
# Windows:
rmdir /s %APPDATA%\PyKnowledge

# macOS / Linux:
rm -rf ~/.local/share/PyKnowledge        # Linux
rm -rf ~/Library/Application\ Support/PyKnowledge  # macOS
```

**注意：此操作将永久删除所有笔记和附件，操作前请导出备份。**

#### Step 4：删除项目源码

```bash
rm -rf Py_Project/
```

### 仅卸载项目但保留数据

```bash
# 仅执行 Step 1 和 Step 4
# 保留 %APPDATA%\PyKnowledge 目录
# 之后重新 pip install 即可恢复使用
```

## 7. 常见问题

### Q1：`pip install` 报 SSL 错误

```
SSLError: [SSL: CERTIFICATE_VERIFY_FAILED]
```

**原因：** 企业网络防火墙拦截 SSL 连接。

**解决：**
- 使用离线安装方式（见第 3 节）
- 或在 IT 部门获取合法代理后配置 pip 代理：
  ```bash
  pip install --proxy=http://proxy.company.com:8080 -e .
  ```

### Q2：`ImportError: No module named 'PySide6'`

**解决：**
```bash
pip install PySide6>=6.6
```

### Q3：启动时报 `setHighDpiScaleFactorRoundingPolicy must be called before...`

**原因：** Qt6 API 顺序要求。已在 `app.py` 中修复，确保使用的是最新代码。

### Q4：`AttributeError: type object 'PySide6.QtWidgets.QTextEdit' has no attribute 'WrapMode'`

**原因：** Qt6 中 `WrapMode` 枚举从 `QTextEdit` 移到了 `QTextOption`。已在 `editor_widget.py` 中修复。

### Q5：Windows 终端输出乱码或 `UnicodeEncodeError`

**原因：** Windows GBK 终端不支持 Emoji 字符。

**解决：** 不影响功能，或使用 Windows Terminal（UTF-8 支持）。

### Q6：如何备份数据？

```bash
# 复制整个应用数据目录即可
# Windows:
xcopy /s %APPDATA%\PyKnowledge D:\Backup\PyKnowledge_Backup\

# 或使用导出功能 → File → Export → 导出为 Markdown 文件夹
```

### Q7：如何迁移到另一台机器？

1. 在新机器上安装 PyKnowledge
2. 将旧机器的数据目录复制到新机器对应位置
3. 启动应用 — 所有笔记、标签、版本、附件完整迁移

### Q8：PySide6 安装太慢或失败

```bash
# 使用国内镜像（如果网络通）
pip install PySide6 -i https://pypi.tuna.tsinghua.edu.cn/simple

# 或只安装最小依赖
pip install PySide6-Essentials
```
